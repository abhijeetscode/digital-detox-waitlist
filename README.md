# Digital Detox App Waitlist

A bold, modern landing page + waitlist for **NFC Social Media Blocker**. Built for GitHub Pages with a single HTML file.

## Preview
- Hosted on GitHub Pages (static)
- Tailwind via CDN
- Email-only waitlist
- Inspirational toast on successful signup

## Tech
- HTML + Tailwind CSS

## Quick Start
1. Open `index.html` in your browser to preview locally.
2. Deploy the repo to GitHub Pages.

## App Info
NFC Social Media Blocker is an Android app that uses NFC taps and quick math challenges to create intentional, 1-minute access windows for social media apps. It includes fullscreen blocking, anti-bypass protections, and encrypted NFC tag registration to break endless scrolling loops.
